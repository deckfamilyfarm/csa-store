# CSA Store Template

This folder contains a static HTML/CSS template you can open directly in a browser.
The sections are modular and can be moved around by cutting the section blocks in index.html.

Files
- index.html: Page layout with clear component boundaries.
- styles.css: Theme tokens and component styles.

How to test
1) Open index.html in a browser.
2) Reorder sections by moving the blocks marked with "component" comments.
3) Edit colors and type in :root inside styles.css.

Theme notes
- Visual goal: Lands' End meets farm.
- Warm, durable palette, catalog clarity, and a handmade texture feel.

Component map
- Announcement bar
- Header + nav
- Hero
- Pantry categories
- CSA plans
- Seasonal highlight
- Product grid
- Herdshare and credits
- Delivery and pickup
- Recipes and stories
- Footer
