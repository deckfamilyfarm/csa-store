# Vite React Template

This folder contains a minimal Vite + React scaffold with separate guest and member routes.

Routes
- /guest shows the plan-first flow.
- /member shows the pantry-first flow plus account settings.
- /page/* routes show individual sections (home, plan-chooser, pantry, csa-plans, seasonal, products, herdshare, delivery, product, recipes, account).

Files
- src/components: modular sections you can reorder in Storefront.jsx.
- src/data.js: content and stock image URLs (Unsplash).
- src/styles.css: shared theme and layout styles.

Run
1) npm install
2) npm run dev
